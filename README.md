# Internship Application Form

## 📌 Project Overview
This is a simple Internship Application Form created using HTML. The form allows applicants to enter their personal details, choose their desired internship role, select their skills, and provide feedback.

## ✨ Features
- Upload profile photo
- Enter personal details
- Choose internship role
- Select technical skills
- Message and feedback section
- Simple and beginner-friendly HTML project

## 🛠️ Technologies Used
- HTML5

## 📂 Project Structure
```
index.html
README.md
```

## 🚀 How to Run
1. Download or clone this repository.
2. Open the `index.html` file in any web browser.
3. Fill in the application form.

## 📸 Preview
Open `index.html` in your browser to view the Internship Application Form.
